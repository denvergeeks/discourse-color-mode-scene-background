# Changelog

All notable changes to this project will be documented in this file.

## 0.5.0 - 2026-06-20

- Switched to a connector-based `.gjs` outlet render path for `above-main-container`.
- Fixed the `didInsert` modifier import path for current Discourse compatibility.
- Narrowed the glass styling to a safer allowlist of core high-value surfaces.
- Added uninstall guidance, Meta publishing notes, and a QA checklist.
- Prepared the repository for public publishing with license and topic draft files.

## 0.4.0 - 2026-06-20

- Removed the unused initializer directory.
- Cleaned the package structure and README guidance.

## 0.3.0 - 2026-06-20

- Added scene-aware glass panel styling for the main UI surfaces.
- Tuned day/night scene transitions and motion controls.

## 0.2.0 - 2026-06-20

- Fixed render modifier import compatibility for current Discourse builds.

## 0.1.0 - 2026-06-20

- Initial release with day/night SVG scenes, automatic mode detection, and configurable motion settings.
