# Screenshots

Add up to two screenshots for Discourse theme/component previews:

- `light.png`
- `dark.png`

Recommended guidance based on current Discourse theme metadata docs:

- Put screenshots in this `screenshots/` folder.
- Use a 16:9 ratio.
- Keep each file under 1 MB.
- Stay at or below 3840×2160.
- PNG or JPEG are safe choices.

Suggested capture set:

- `light.png`: homepage or latest view in light mode with header, sidebar, and topic list visible
- `dark.png`: same composition in dark mode
